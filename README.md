## 2. 企画概要 & 解決する課題

「フィーリングコレクション」は、**スマホで顔を判別し、手持ちの服やその日の気分・気候・シーンに合わせた最適なコーディネートを提案・アドバイスするiPhoneアプリ**。

### 💡 背景と差別化ポイント（先行サービスとの違い）

単に服を検索するアプリとは異なり、以下の3つの強みでユーザーの「服選びの悩み」を解決します。

1. **「顔タイプ判別 × 手持ち服」の掛け合わせ：** 自分の顔の系統（子供顔/大人顔、直線/曲線など）に合ったアドバイスに加え、手持ちの洋服に合う組み合わせを提案。
2. **気分・気候・シーンに完全連動：** その日の天気（気温・湿度）や「今日は仕事でしっかり」「休日にリラックス」といった気分やシーンに合わせた服を提案。
3. **提案から購入までがシームレス：** 足りないアイテムや気になった洋服は、アプリ内からすぐに購入サイト（URL）へアクセス可能。

```
  ▼ 『フィーリングコレクション』を利用すると…

【After】

・手持ちの服を生かしつつ、足りない服も賢く購入できるようになる！

```

## 5. 機能一覧

| 区分 | 機能名 | 内容 |
| ---  |  ---  |  --- |
| **Must**

*(必須開発)* | **自分の顔と洋服の合成機能** | カメラ映像・画面上で自分の顔と洋服画像を重ね合わせて確認できる機能 |
|  | **顔タイプ判別機能** | インカメラから顔のパーツ特徴を解析し、タイプを判定する機能 |
|  | **気候連動提案機能** | 気温や湿度のデータ（API）を取得し、快適な服を自動提案する機能 |
|  | **洋服登録機能（クローゼット）** | 自分の手持ちの洋服をアプリ内に画像・データとして保存する機能 |
|  | **ECサイトURL取得機能** | 提案された服や合わせたい服がすぐに買える購入ページへのリンク機能 |
| **Want**

*(追加機能)* | **リアルタイム試着機能** | カメラ越しにリアルタイムで顔と洋服を合わせられるAR風UI |
|  | **顔系統別アドバイス機能** | 判定された顔タイプに基づき、おすすめの着こなしアドバイスを表示 |
|  | **小物（アクセサリー）合成** | ピアスやネックレスなどの小物も合わせてコーディネートできる機能 |
